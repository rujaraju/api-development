API Timestamp Microservice

Please pass a string as a parameter, and this service will check to see whether that string contains either a unix timestamp or a natural language date (example: January 1, 2016)

  • If it does, it returns both the Unix timestamp and the natural language form of that date.
  • If it does not contain a date or Unix timestamp, it returns null for those properties.

Example Usages:

https://api-developments-ruja.c9users.io/timestamp/December%2015,%202015

https://api-developments-ruja.c9users.io/timestamp/1450137600

Example Output:

{ "unix": 1450137600, "natural": "December 15, 2015" }
